🎯 САМЫЙ ПРОСТОЙ СПОСОБ:
═══════════════════════════════════════════════════════════════

1. Войдите в WordPress Admin
2. Откройте пост мотоцикла
3. В правой панели → "Избранное изображение"
4. Загрузите фото
5. Нажмите "Обновить"

✅ ГОТОВО!

───────────────────────────────────────────────────────────────

📁 РЕЗЕРВНОЕ ИЗОБРАЖЕНИЕ:
═══════════════════════════════════════════════════════════════

Если нужно изображение по умолчанию для всех мотоциклов:

→ Поместите файл сюда:
  /wp-content/themes/badboybike/images/

→ Назовите точно так:
  default-motorcycle-hero.jpg

✅ Изображение появится автоматически!

───────────────────────────────────────────────────────────────

✓ Формат:     JPG (для фото)
