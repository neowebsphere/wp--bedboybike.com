📋 БЫСТРАЯ ИНСТРУКЦИЯ:

╔═══════════════════════════════════════════════════════════╗
║  СПОСОБ 1: Через WordPress (Рекомендуется)               ║
╚═══════════════════════════════════════════════════════════╝

1. Войдите в админ-панель WordPress
   → http://ваш-сайт.com/wp-admin

2. Перейдите к посту мотоцикла
   → Записи → Motorcycles (или ваш тип записи)
   → Нажмите на нужный мотоцикл

3. Установите избранное изображение
   → В правой панели найдите "Избранное изображение"
   → Нажмите "Установить избранное изображение"
   → Загрузите фото или выберите из библиотеки
   → Нажмите "Установить избранное изображение"

4. Сохраните изменения
   → Нажмите "Обновить"

✅ ГОТОВО! Изображение появится на странице мотоцикла

---

╔═══════════════════════════════════════════════════════════╗
║  СПОСОБ 2: Резервное изображение по умолчанию            ║
╚═══════════════════════════════════════════════════════════╝

Если не установлено избранное изображение, система 
автоматически использует:
   📁 /images/default-motorcycle-hero.jpg

КАК ДОБАВИТЬ:

1. Подготовьте изображение:
   • Размер: 1920x1080 пикселей (минимум)
   • Формат: JPG или PNG
   • Вес: до 800 КБ (оптимизируйте!)

2. Переименуйте файл:
   → default-motorcycle-hero.jpg

3. Поместите файл в папку:
   → /wp-content/themes/badboybike/images/

✅ Изображение появится автоматически для всех мотоциклов
   без установленного избранного изображения!

╔═══════════════════════════════════════════════════════════╗
║  📸 РЕКОМЕНДАЦИИ ПО ИЗОБРАЖЕНИЯМ                         ║
╚═══════════════════════════════════════════════════════════╝

✓ Оптимальный размер: 1920x1080 пикселей
✓ Соотношение сторон: 16:9 (альбомная)
✓ Формат: JPG (для фото), PNG (для графики)
✓ Размер файла: 300-800 КБ
✓ Качество: Высокое разрешение, четкие детали

КОМПОЗИЦИЯ:
✓ Мотоцикл по центру или по правилу третей
✓ Хорошее освещение
✓ Чистый фон без лишних деталей
✓ Высокий контраст между байком и фоном

ИЗБЕГАЙТЕ:
✗ Размытых или низкого качества фото
✗ Вертикальной ориентации
✗ Водяных знаков
✗ Файлов больше 2 МБ

ПРОБЛЕМА: Изображение не отображается
→ ✓ Проверьте имя файла: default-motorcycle-hero.jpg
→ ✓ Убедитесь, что файл в папке /images/
→ ✓ Очистите кэш WordPress
→ ✓ Проверьте права доступа к файлу (должно быть 644)

ПРОБЛЕМА: Изображение растянуто
→ ✓ Используйте соотношение сторон 16:9
→ ✓ Минимальная ширина: 1920px

ПРОБЛЕМА: Медленная загрузка
→ ✓ Сожмите изображение (до 800 КБ)
→ ✓ Используйте JPG вместо PNG для фото
→ ✓ Рассмотрите формат WebP

ПРОБЛЕМА: Изображение слишком темное
→ ✓ Hero секция добавляет тёмный градиент автоматически
→ ✓ Используйте более светлые/яркие фото
→ ✓ Можно настроить градиент в CSS

Hero секция автоматически:
✅ Использует избранное изображение из WordPress
✅ Показывает резервное изображение, если основное не установлено
✅ Применяет parallax-эффект при скролле
✅ Добавляет тёмный градиент для читабельности текста
✅ Адаптивна для всех устройств (ПК, планшеты, телефоны)
✅ Анимирует элементы при загрузке страницы
